### 📌 Getting Started!
---
→ Setting up the notify channel! `/anime admin setchannel`<br /><br />
→ Adding Anime `/anime add`<br />
→ Anime List `/anime list`<br />
→ Subscribing `/anime subscribe`<br />
→ That's all. Enjoy!
